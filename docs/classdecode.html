Detailed Description

Class which implements decoding object which contains functions for decoding gcode.

This class allows us to read a line of gcode, decode it, and store the necessary information into class member data. This is a convenient way to store and pass along information as it updates with each new command.

XYSFvalues decode::get_XYSF ( void  )

Get-er functions:

Function which gets the X Y S and F values from the gcode decoder class.

This function gets the struct _XYSFval from the class member data in order to use it in the translator. _XYSFval contains 4 variables; X (desired X position), Y (desired Y position) , S (desired laser PWM value) and F (desired feedrate).
