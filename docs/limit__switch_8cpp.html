void task_debouncer ( void *  p_params)

Task which runs a debouncer for a button or a switch.

This task uses a Constructor called Debouncer to set up a class called blue_button in order to filter out noise that can be caused when pushing a button. This program is simulated by hand using the blue button (PC13) on the Nucleo at a task called every 500 ms or so. When the task is run quicker (at say 10ms) the blue button behaves very much like a button should.
