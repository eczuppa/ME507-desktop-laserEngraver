Class which implements a P, PI, or PID controller for a motor control.

Sets up the P, PI, and PID controller classes and sets up the methods for each of these controller classes.

Creates a PID controller to drive a motor

This constructor sets up the PID controller. It does this by taking in two gain values and a setpoint (gotten from the GCode). It reads the actual position from the encoder output, compares this to the theoretical position from the setpoint (initially from the GCode), computes an error value and updates the setpoint (motor command) to minimize the error. It also will have functions that allow the gain and setpoint protected variables to be changed by either the user or by the GCode.
