class  Kinematics_coreXY
 Class which contains functions to convert from cartesian global coordinates to motor commands. More...
 

Detailed Description

the interface for the kinematics class that wil translate GCODE inputs into a usable type for the control loop

Uses the appropriate kinematic equations to relate Gcode X,Y positions and F feedrate to values of the same type as the motor encoders output. This transformation significantly increases the ease of controlling the motors and only requires the proper transformation of the desired target positions, velocities, etc. from the Gcode. CoreXY concepts used from Ilan E. Moyer - one of the earlier authors of CoreXY implementation and theory his website is available here: https://corexy.com/theory.html
