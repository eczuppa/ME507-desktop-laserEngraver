void encoder_A_task ( void *  p_params)

Task to run the left encoder, Encoder A (Motor 1 Encoder)

This task will interface with the Quad_Encoder class in order to read values from Encoder A. It will use methods established in the Quad_Encoder class to send values, via queues, of position, velocity, and delta time to the controller.
