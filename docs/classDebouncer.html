Class which implements a debouncer for button and switch noise.

Mostly defines the function "update" and impliments the class into the constructor

Creates a debouncer to filter through noise from buttons and switches

This contructor looks at a data stream, identifies when the a signal has been activated, and switches state. It sets up an State Machine using a set of IF statements to determine what the values are of an input pin, counts how many consecutive 0's or 1's are detected, and what corresponding state it should be in. State 0 outputs a FALSE and State 2 outputs a TRUE.

Parameters
pinThis is the parameter of which pin our button or switch is connected to
thresholdThis is set as whatever we want our threshold to be for how many consecutive 1's or 0's are acceptable for saying the button or switch is active or not.
