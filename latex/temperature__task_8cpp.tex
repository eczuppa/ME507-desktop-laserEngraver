implements a task function to use digital temperature sensor like an industrial heat detector 

uses the One\+Wire library and Dallas\+Temperature library to interface with the a D\+S18\+B20 digital temperature sensor that is looking at the rate of rise in temperature over a sustained period of time. This task sets up and runs the temperature sensor in this configuration and queues the data to the safety supervisor task function.
