File containing the interface for the kinematics class that wil translate G\+C\+O\+DE inputs into a usable type for the control loop. 

This file contains a class which uses the appropriate kinematic equations for Core\+XY to relate Gcode X,Y positions and F feedrate to values of the same type as the motor encoders output.
