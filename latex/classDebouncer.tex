Creates a debouncer to filter through noise from buttons and switches. 

This contructor looks at a data stream, identifies when the a signal has been activated, and switches state. It sets up an State Machine using a set of IF statements to determine what the values are of an input pin, counts how many consecutive 0\textquotesingle{}s or 1\textquotesingle{}s are detected, and what corresponding state it should be in. State 0 outputs a F\+A\+L\+SE and State 2 outputs a T\+R\+UE.
